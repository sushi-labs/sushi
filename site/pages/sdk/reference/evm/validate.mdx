# Reference: `sushi/evm` Validate

The EVM validation module exposes zod helpers for EVM values.

## Exports

| Export | Purpose |
|---|---|
| `evmAddress` | Zod schema factory for EVM addresses. |
| `szevm` | EVM schema namespace. Currently includes `szevm.address`. |

```ts twoslash
import { evmAddress, szevm } from 'sushi/evm'

const address = evmAddress().parse(
  '0xA0b86991c6218b36c1d19D4a2e9Eb0cE3606eB48',
)

const same = szevm.address().parse(
  '0xA0b86991c6218b36c1d19D4a2e9Eb0cE3606eB48',
)
```
